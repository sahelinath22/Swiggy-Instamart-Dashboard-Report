# Swiggy-Instamart-Dashboard-Report
In this dashboard, I explored and visualized key insights to understand the business performance and growth trends of Swiggy Instamart.  Using DAX formulas, cards, charts, and slicers, I presented valuable insights such as:  ✅ Total Sales, Orders &amp; Ratings  ✅ City-wise and Shop-size-wise Sales &amp; so on.
